# 美德闯关赛｜初中道法 AI 互动网页

由 Andyyu001 设计，用于《传承中华传统美德》课堂教学环节五“美德闯关赛”，支持：
- 5道选择题闯关
- 提交后自动判分
- **DeepSeek AI 生成错题讲解**（初中生友好语言）
- 语音播报 + 图片嵌入（第5题配图）

## 🚀 部署方式
1. 将本仓库部署到 [Netlify](https://www.netlify.com/)
2. 在 Netlify 设置环境变量：`DEEPSEEK_API_KEY=sk-xxx`
3. 上传图片至 `img/q5.jpg`
4. 完成！
